# Etch-a-Sketch

A browser-based sketchpad built with **HTML, CSS, and JavaScript**.

Users can draw by simply hovering over the grid, resize the canvas, and enjoy a simple pixel-art drawing experience.

## Features

- 16 × 16 grid generated dynamically using JavaScript
- Hover to draw on the canvas
- Resize the grid (up to 100 × 100)
- Grid automatically fits inside the same drawing area
- Responsive layout using **Flexbox**
- Clean and minimal UI

### Extra Features (if implemented)

1. Random RGB color on every hover
2. Progressive darkening effect (10% darker per interaction)
3. Clear/Reset grid button

## Getting Started

### Clone the repository

git clone https://github.com//etch-a-sketch.git

### Open the project

Simply open `index.html` in your browser.

No additional installation is required.

## How to Use

1. Move your mouse over the grid to draw.
2. Click **Resize Grid**.
3. Enter a value between **1 and 100**.
4. A new grid is generated while keeping the drawing area the same size.
